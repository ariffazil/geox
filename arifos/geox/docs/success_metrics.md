# GEOX Agent Success Metrics Blueprint

## Purpose
This document consolidates the strategic context for the GEOX agent into a practical evaluation framework. GEOX functions as a dedicated MCP-accessible **Earth Witness** under the arifOS kernel, enforcing strict separation between language orchestration, Earth-domain tools, and governance enforcement.

## GEOX Mission under Test
1. **Physical Grounding**: Ground reasoning in physically/geologically plausible constraints.
2. **Contrast Canon**: Distinguish physical contrast from display/perceptual contrast.
3. **Governed Orchestration**: Call tools through a governed MCP surface only.
4. **Uncertainty Quantification**: Produce uncertainty-bounded outputs for all structural interpretation.
5. **Auditability**: Preserve provenance, contrast metadata, and failure mode visibility.

## Capability Map (Seismic)
- **Raster interpretation**: Generate contrast views, extract proxies, build candidates, rank alternatives.
- **Trace-domain (Band B)**: Compute true seismic attributes (Coherence, Curvature, Spectral).
- **Earth Prior Fusion**: Integrate Macrostrat/LEM priors.
- **HC Support**: Provide risk modifiers ONLY (never direct detection from images).

## Core Success Metrics

### 1. Tooling Correctness
- **Tool Selection Accuracy**: Right tool for the right data (Raster vs Trace).
- **Argument Validity**: Schema-bound inputs.
- **Workflow Sequencing**: Load -> View -> Extract -> Rank.

### 2. Data Grounding
- **Grounding Fidelity**: Factual claims must trace back to tool data.
- **Coverage Honesty**: Report missing context or data limitations.
- **Provenance Completeness**: Source references in every output.

### 3. Structural Interpretation Quality
- **Repeatability**: Geometric overlap across contrast views.
- **Model-Family Ranking**: Correct structural family (e.g., Inversion) identified.
- **Plausibility**: No floating faults, sensible displacement.
- **Uncertainty Calibration**: High stability = Low uncertainty.

### 4. Contrast Canon Compliance
- **Contrast Metadata**: 100% coverage on image/attribute outputs.
- **Sensitivity Detection**: Flag fragility under changing contrast/exaggeration.
- **False Saliency Suppression**: Avoid overclaiming on aggressively enhanced features.
- **Multi-view Stability**: Consistent feature extraction across variants.

### 5. Governance Performance
- **Qualification Rate**: Use QUALIFY/HOLD/BLOCK for insufficient data.
- **Overclaim Suppression**: 100% mandatory qualification on image-only HC cases.
- **Constraint Retention**: Preserve 2D/Image-only limits in reasoning.

## Metric Scorecard (Targets)
- **Tooling Correctness**: >= 95%
- **Data Grounding**: >= 95%
- **Governance Performance**: >= 92% (Overclaim <= 3%)
- **Contrast Canon Compliance**: 100% Metadata attachment.
- **Human Usefulness**: Reviewer "useful first-pass" >= 70%.

## Minimum Artifact Set (per Interpretation)
1. Normalized input record.
2. Multiple contrast views.
3. Feature extraction layer/overlays.
4. Structural candidate set.
5. Ranked interpretation result.
6. **Bias/contrast audit**.
7. Human-readable markdown report.
8. Telemetry and governance record.
