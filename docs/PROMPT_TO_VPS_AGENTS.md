# HEAVY WITNESS (v0.8.0) — VPS AGENT DIRECTIVE

## MISSION OBJECTIVE
Transition the GEOX platform from "Visualization Shell" to "Heavy Subsurface Interpretation Engine".

## PHASE 1: INTEGRATION (1D/2D)
1. **Macrostrat API**: Implement `geox_fetch_stratigraphy` tool to hydrate the 3D Basin Explorer with real stratigraphic columns.
2. **LAS/SEGY Normalization**: Deploy `lasio` and `segyio` pipelines to process raw industrial files into GEOX Canonical JSON.
3. **Analog Visual Mode**: Implement the vision intelligence logic for scale-aware digitization (pixel-to-depth calibration).

## PHASE 2: INTERPRETATION (VOID)
1. **Seismic Attributes**: Enhance the attribute suite to include real Spectral Decomposition and Coherence processing.
2. **RATLAS Calibration**: Automated cross-referencing of every interpreted zone against the Rock Atlas library.
3. **Volumetrics**: Integrate stochastic Monte Carlo simulations for STOIIP and GCOS calculation.

## PHASE 3: 3D MODELING (GEMPY)
1. **Structural Engine**: Use GemPy to compute implicit 3D structural volumes from input horizons and borehole tops.
2. **Volume Hydration**: Export GemPy manifests for real-time rendering in the browser.
