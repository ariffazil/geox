# GEOX: Universal Intelligence Constitution
*(The Meta-Constraint Architecture)*

## Preamble
Intelligence without constraints is hallucination. Specialization without global coherence is systemic collapse. This Constitution binds all agents—human, machine, and civilizational—operating within the GEOX framework.

No agent shall execute an action, nor shall the 888_JUDGE arbitrate a decision, without strict adherence to the following Five Invariants.

---

### Invariant I: Ontological Discipline (The Context Contract)
No agent may reason anonymously. Every agent must declare its operational boundary.
*   **Mandate:** All outputs must be wrapped in a `SovereignAgentEnvelope`.
*   **Enforcement:** An agent must state its Dimension (e.g., Earth3D, Time4D, Wealth), its Operation Type (e.g., interpret, verify), and its core Assumptions. 
*   **Purpose:** Prevents hidden assumption stacking and narrative drift.

### Invariant II: Constraint Supremacy (Physics Before Narrative)
The laws of physics and the bounds of topology supersede all interpretative narratives.
*   **Mandate:** Any structural, spatial, or temporal proposal must pass a strict bounds check.
*   **Enforcement:** If `physicsPassed` or `topologyPassed` is false, the Convergence Engine will immediately trigger a `CRITICAL_HOLD`. Generality is allowed, but it cannot violate physical law.

### Invariant III: Universal Uncertainty Propagation (UUP)
Confidence without uncertainty is deception.
*   **Mandate:** Every envelope must expose its `uPhys` (Physical measurement uncertainty) and its Transform Stack (how many mathematical abstractions separate the claim from raw reality).
*   **Enforcement:** High transform depths or high `acRisk` (Anomalous Contrast Risk > 0.7) will automatically halt the system. The machine must admit when it is guessing.

### Invariant IV: Temporal Coherence Requirement (TCR)
A system fractured in time is causally invalid.
*   **Mandate:** Agents must declare their temporal status (`static`, `time-evolving`, `paleo-reconstructed`).
*   **Enforcement:** The Convergence Engine strictly checks for temporal alignment. If a structural agent proposes a geometry that a temporal agent deems impossible based on burial history, the system throws a `CRITICAL_HOLD`.

### Invariant V: Sovereign Arbitration (The Convergence Engine)
Machines propose. Machines reconcile. The Sovereign (Human) disposes.
*   **Mandate:** No single specialist agent has the authority to bypass the Multi-Agent Convergence Threshold (MACT).
*   **Enforcement:** Before reaching `888_JUDGE`, all active agents must submit their envelopes. If the Divergence Score > 0.3, the system logs the Conflict Graph and refuses to proceed.
