## I. The Three Layers

### Layer 1: LLM (Large Language Model)

**What it is:** A fluent reasoning surface trained on human text.

**What it does well:**
- Natural language understanding
- Synthesis across domains
- Explanation and education
- Pattern recognition in text

**What it fails at (alone):**
- Physical grounding (does the Earth actually work this way?)
- Spatial verification (are these coordinates valid?)
- Uncertainty calibration (how confident should I be?)
- Authority boundaries (when should I stop and ask for help?)

**The core problem:** LLMs optimize for **plausibility**, not **truth**. In geoscience, this is dangerous.

> *"A plain LLM can describe a trap beautifully. It cannot tell you whether the trap is actually grounded."*

### Layer 2: LEM (Large Earth Model)

**What it is:** A physics-grounded Earth knowledge system — **GEOX**.

**What it adds:**
- **Deterministic physics** — EARTH.CANON_9 variables
- **Spatial grounding** — Maps, coordinates, CRS verification
- **Material intelligence** — RATLAS (99 canonical materials)
- **Interpretation discipline** — Theory of Anomalous Contrast
- **Uncertainty quantification** — Confidence bands on every output

**The core insight:** Earth interpretation is not a language problem. It is a **physics + evidence + uncertainty** problem.

**GEOX is the LEM.** It provides what LLMs lack:

| LLM Gap | LEM Solution |
|---------|--------------|
| No coordinate verification | `geox_verify_geospatial` — validates WGS84, CRS, jurisdiction |
| No physics constraints | `geox_feasibility_check` — thermodynamic + mechanical validation |
| No material knowledge | RATLAS — 99 materials with forward models |
| No uncertainty bounds | F7 Humility — Ω₀ ∈ [0.03, 0.15] enforced |
| No display artifact awareness | ToAC — prevents anomalous contrast errors |

### Layer 3: arifOS (Constitutional OS)

**What it is:** A governance layer that decides what grounded systems are allowed to claim.

**What it adds:**
- **13 Binding Floors** — Constitutional constraints on all outputs
- **888_JUDGE** — Human sovereignty gates
- **999_VAULT** — Immutable audit trails
- **Trinity Architecture** — Separation of concerns (ΔΩΨ)
- **Metabolic Pipeline** — 000-999 execution stages

**The core insight:** Grounding alone is not governance. You need authority boundaries, failure discipline, and explicit human primacy.

**Problems:**
- No coordinate verification
- No physics check (is this geometry possible?)
- No uncertainty declaration
- No audit trail
- No human gate

