## IV. Page Conventions

### Frontmatter (Required)

Every wiki page MUST include YAML frontmatter:

```yaml
---
type: [Theory | Material | Basin | Tool | Case | Governance | Integration]
tags: [array, of, relevant, tags]
sources: [raw/papers/bond_2007.pdf, raw/basin_reports/malay_basin_2024.pdf]
last_sync: 2026-04-08
confidence: 0.95                    # F7 Humility: explicit confidence
certainty_band: [0.85, 0.99]        # Ω₀ interval
epistemic_level: [OBS | DER | INT | SPEC]
arifos_floor: [F1 | F2 | F4 | F7 | F9 | F11 | F13]  # Primary floor
---
```

### Epistemic Levels (Critical for GEOX)

| Level | Code | Meaning | Example |
|-------|------|---------|---------|
| **OBSERVATIONAL** | OBS | Raw sensor data | Seismic amplitude, log curves |
| **DERIVED** | DER | Computed from observations | Porosity, saturation, volume |
| **INTERPRETED** | INT | Inferred from derived data | Lithology, fluid type, pay |
| **SPECULATED** | SPEC | Proposed but unverified | Prospect risk, OOIP estimates |

**⚠️ F9 Anti-Hantu Violation:** Collapsing levels (treating INT as OBS) triggers automatic 888_HOLD.

### Cross-Reference Format

| Type | Syntax | Example |
|------|--------|---------|
| Internal wiki | `[[Page_Name]]` | `[[Theory_of_Anomalous_Contrast]]` |
| Raw source | `[[raw/filename]]` | `[[raw/papers/bond_2007.pdf]]` |
| arifOS wiki | `[[arifos::Page_Name]]` | `[[arifos::Floors]]` |
| Specific section | `[[Page_Name#Section]]` | `[[Floors#F4_Clarity]]` |

### Naming Conventions

| Category | Format | Example |
|----------|--------|---------|
| Theory pages | Title_Case_With_Underscores | `Theory_of_Anomalous_Contrast.md` |

## V. Operations

### 1. INGEST — Adding New Knowledge

**Trigger:** New source added to `raw/`

**Workflow:**
```
1. Human adds source to raw/ (paper, report, log, etc.)
2. LLM reads source + discusses key takeaways with human
3. LLM writes summary page in wiki
4. LLM updates relevant entity/concept pages (10-15 pages typical)
5. LLM updates index.md with new entry
6. LLM appends to log.md with consistent format
7. Human reviews changes in Obsidian (or similar)
```

**Log Entry Format:**
```markdown
## [2026-04-08] ingest | Bond et al. 2007 — Cognitive bias in seismic interpretation
- Source: [[raw/papers/bond_2007.pdf]]
- Pages created/updated:
  - [[Bond_2007_Cognitive_Bias]] (new)
  - [[Theory_of_Anomalous_Contrast]] (updated: added citation)
  - [[60_CASES/Structural_Interpretation_Workflow]] (updated: bias audit step)
- Key findings: 79% expert failure rate on synthetic seismic with display artifacts
- Contradictions flagged: None
- Next actions: Add Bond 2007 reference to all structural interpretation cases
```

### 2. QUERY — Answering Questions

**Trigger:** User asks geoscience question

**Workflow:**
```
1. LLM reads index.md to find relevant pages
2. LLM drills into specific pages (follows links)
3. LLM synthesizes answer with explicit citations
4. LLM declares epistemic level of answer
5. LLM suggests filing valuable synthesis back into wiki
```

**Output Format:**
```markdown
**Answer:** [Synthesized response with inline citations like [[Page_Name]]]

**Epistemic Level:** [OBS/DER/INT/SPEC]
**Confidence:** [Ω₀ value] (F7 Humility)
**Sources:** [[Page1]], [[Page2]], [[raw/source.pdf]]

**Suggested Actions:**
- File this synthesis as [[New_Page_Name]]
- Link from [[Related_Concept]]
```

### 3. LINT — Health Check

**Trigger:** Weekly, or on demand

**Checks:**
- [ ] **Contradictions** — Do any pages contradict each other?
- [ ] **Stale claims** — Has new data superseded old claims?
- [ ] **Orphan pages** — Pages with no inbound links?
- [ ] **Missing pages** — Important concepts mentioned but not created?
- [ ] **Missing cross-references** — Mentions without links?
- [ ] **Data gaps** — What sources could fill known gaps?
- [ ] **Epistemic drift** — Any INT/SPEC presented as OBS/DER?
- [ ] **Floor compliance** — F4 (scale/CRS), F7 (confidence), F11 (provenance)

**Lint Report Format:**
```markdown
## Lint Report — 2026-04-08

### Contradictions Found
| Page 1 | Page 2 | Issue | Resolution |
|--------|--------|-------|------------|
| [[Porosity_Types]] | [[60_CASES/Case_2026_03]] | φt vs φe usage | Update case to specify φt |

### Orphan Pages
- [[Elastic_Moduli]] — no inbound links. Suggest linking from [[EARTH_CANON_9]].

### Missing Pages (High Priority)
- Simandoux equation mentioned in 3 pages but no dedicated page
- Suggest creating [[Simandoux_Equation]]

- Suggest searching: "Indonesia heat flow basin analysis 2020+"
```
